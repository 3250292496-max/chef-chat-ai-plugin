# 厨师聊天AI插件

这是一个全新命名空间的 OlivOS/OlivaDice AI 聊天插件，不复用旧的 AI 聊天插件名称。

## 主要功能

- 在菜单“厨师聊天AI插件设置”里配置 DeepSeek、OpenAI 或其他 OpenAI 兼容接口。
- OpenRouter 预设默认使用 `https://openrouter.ai/api/v1` 和 `openai/gpt-5.5`；填入 OpenRouter API Key 后即可测试。附加请求头里的 `HTTP-Referer` 和 `X-OpenRouter-Title` 是 OpenRouter 可选项，不填也能请求。
- 支持 API、人设、资料库、聊天功能分别保存，底部也有全局保存。
- 支持导入/导出 API 配置、人设文本和常驻资料文件。
- 资料库按文件保存：原文件保存在 `plugin/data/ChefChatAI/knowledge_files/sources`，解析后的文本缓存保存在 `plugin/data/ChefChatAI/knowledge_files/texts`，索引保存在 `plugin/data/ChefChatAI/knowledge_files/index.json`。
- 支持导入 `txt`、`md`、`json`、`csv`、`docx` 等文件。AI 每次回复时会按问题检索启用资料文件，只注入相关片段，可用于模组设定、个人世界观、人物资料、地点资料等。
- 支持环境感知：AI 可在提示词中获得本机当前日期时间；可选注入运行环境摘要。
- 支持 RSS 新闻感知：用户询问新闻、热点、头条等内容时，插件会抓取新闻 RSS 标题/摘要，并尽量读取原文正文摘录后注入给 AI；也可以直接用本地指令抓取新闻内容。
- 支持网络眼睛：用户明确要求搜索、查询、最新信息、抓取网页或发送网址时，插件会联网检索公开网页，读取可用摘录，并按来源、要点和结论慢慢整理给 AI。
- 支持情绪图片：AI 回复时可输出隐藏情绪标记，插件会移除标记并从本地情绪图片目录随机发送对应图片。
- 支持设置机器人主人 QQ，插件只把“当前发言人是否为主人”告诉 AI，不向模型公开真实 QQ。
- API 导出时会询问是否包含 API Key；默认可导出不含密钥的配置模板。
- 支持三种回复风格：简短群聊、详细设定、创作润色，可在设置页选择，也可由主人用指令切换。
- 支持活人感功能：AI 可用隐藏停顿符让插件分多条短消息连续发送，插件会自动合并太短或像半个词的片段，并把情绪图片穿插在回复中间。
- 支持群聊自然插话：普通群聊消息会先按问句、情绪、玩梗和触发词评分，再经过概率、冷却和每小时上限；合适时可以像群友一样短句接话，也可以从表情包库挑一张合适的表情。
- 支持低频自动偷图：群友发图时先本地筛选、去重、过概率和冷却，只有通过筛选的新图才调用识图 API 自动命名并放进表情包库。
- 支持群聊关键词触发、@ 机器人触发、私聊触发、上下文记忆、状态查询和清空上下文。
- 主配置保存在 `plugin/data/ChefChatAI/config.json`，历史记录保存在 `plugin/data/ChefChatAI/history.json`。

## 默认触发

- 群聊：`厨师 你的问题`、`.chef 你的问题`、`.ai 你的问题`，或 @ 机器人后提问。短英文触发词默认需要空格、冒号等分隔符。
- 私聊：默认也使用触发词；可在设置里开启“私聊任意消息自动回复”。
- 自然插话：设置页“聊天功能”里可调基础概率、强触发概率、冷却、同人冷却、每小时上限、触发词和自动偷图预算；主人可用 `厨师 插话状态`、`厨师 插话开启`、`厨师 插话关闭` 控制。设置页也支持单独导入/导出插话模块配置。

## 环境感知和网页检索

设置页“环境感知”里可以控制本机时间、RSS 新闻、网页检索、网页读取和检索缓存。

- 新闻感知会先读取 RSS 标题/摘要，再尝试打开新闻原文并抽取正文摘录。
- 网页检索支持 `厨师 搜索 关键词`、`厨师 查一下 关键词`，普通提问里带“最新、当前、网页、链接、资料、扒一下、联网看看、官网”等词时也会尝试注入检索结果。
- 直接搜索指令会分几条发送：先说明检索对象，再列来源，再提炼摘录重点，最后给核对链接；AI 回复也会被提示按来源、要点、结论慢慢说。
- 安全模式默认开启，会拦截本机地址、内网地址和非常见端口；插件只读取公开网页文本，不读取本地文件、密钥或后台隐私。
- 读取到的内容会作为来源线索交给 AI，AI 仍应提醒重要信息以原文或权威来源为准。

## 活人感功能

设置页“聊天功能”里可以调整活人感功能：

- `启用活人感`：开启后 AI 回复可像聊天一样分多条短消息发送。
- `AI 停顿符`：允许模型在自然停顿处插入隐藏标记 `[[split]]`，插件会删除标记并分条发送。
- `自动拆短句`：模型没写停顿符时，插件会按换行、句号、逗号、省略号等自然停顿拆分。
- `最短消息`：太短或像半个词的片段会自动并回前后句，避免把“我们”拆成“我 / 们”这种怪气泡。
- “自然分气泡”开启时，插件会像真人聊天一样拆成几条完整气泡；逗号、顿号、冒号和半句话会自动并回下一条。
- `表情穿插发送` / `AI 图片位置符`：让情绪图片尽量插在短句之间；模型也可用隐藏标记 `[[image]]` 指定图片位置。
- `最多消息`、`每条字符`、`最短消息`、`基础间隔`、`浮动秒`：控制聊天气泡数量和节奏。

默认最多 5 条、基础间隔 1.1 秒。代码块、编号清单和结构化长文会尽量保持一条发送。

## 情绪图片

图片目录会自动创建在 `plugin/data/ChefChatAI/emotion_images` 下。默认子目录包括：

- `happy`：开心
- `sad`：难过
- `comfort`：安慰
- `angry`：生气
- `surprised`：惊讶
- `embarrassed`：尴尬
- `thinking`：思考
- `playful`：玩梗
- `neutral`：平静

把 `.jpg`、`.png`、`.gif`、`.webp` 等图片放入对应目录后，AI 回复时会按情绪随机发送。可在设置页“情绪图片”里调整概率、冷却和开关。
